Detailed Description

Represents the class of an internal memory device of a virtual machine.

The class is for storing char values in an array, whose size is immutable and limited my the max value of the type long long. Also provides an opportunity to protect cells from writing (i.e. locking), the indexes of th locked are stored in a list
